# APRENDIZADO EM C

# Sobre o repositorio

  Este repositório foi criado com o objetivo de documentar minha evolução nos estudos da linguagem C, reunindo anotações, exercícios, desafios e projetos desenvolvidos ao longo da minha jornada de aprendizado. O conteúdo está organizado do nível básico ao avançado, abordando conceitos fundamentais de programação, lógica, estruturas de dados, manipulação de memória, ponteiros, funções, arquivos e outros temas importantes da linguagem.

## Topicos

 - Compilação                 
 - Bibliotecas
 - Variaveis
 - Input/Outpu
 - Operações
 - Estruturas de controle
 - Estruturas de repetição
 - Vetores e Matrizes
 - Strings
 - Tipos definidos pelo programador
 - Ponteiros
 - Alocação de memoria
 - Arquivos
